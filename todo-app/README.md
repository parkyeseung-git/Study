# ToDoApp

명세서(`ToDoApp_명세서.md`) 기반 할 일 관리 웹 앱입니다.

## 실행 방법

1. **로컬 서버로 실행 (권장)**  
   프로젝트 루트에서:
   ```bash
   npx serve todo-app
   ```
   또는 Python:
   ```bash
   cd todo-app && python3 -m http.server 8080
   ```
   브라우저에서 `http://localhost:3000` 또는 `http://localhost:8080` 접속.

2. **파일로 열기**  
   `todo-app/index.html`을 브라우저에서 직접 열어도 동작합니다. (로컬 스토리지는 동일 도메인에서만 유지됩니다.)

## 구현된 기능 (명세서 대비)

| 구분 | 기능 | 상태 |
|------|------|------|
| F-01 | 할 일 추가 | ✅ |
| F-02 | 할 일 목록 조회 (최신순) | ✅ |
| F-03 | 완료/미완료 전환 | ✅ |
| F-04 | 할 일 수정 (더블클릭 또는 수정 버튼) | ✅ |
| F-05 | 할 일 삭제 | ✅ |
| F-06 | 필터: 전체 / 진행 중 / 완료 | ✅ |
| F-09 | 로컬 스토리지 데이터 유지 | ✅ |
| F-10 | 완료 항목 일괄 삭제 | ✅ |

- 반응형 레이아웃, 키보드·포커스 접근성 반영
- 데이터 모델: `id`, `title`, `completed`, `createdAt` (명세서 4장)
