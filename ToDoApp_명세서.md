# ReBorn(ToDoApp) 웹 앱 명세서

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | ReBorn |
| 목적 | 일정 중심의 개인 할 일 관리 (추가/반복/기간/완료/복원) |
| 대상 | 데스크톱·모바일 브라우저 사용자 |
| 기본 진입 화면 | ReBorn 탭(`my`) |

---

## 2. 화면/탭 구성

### 2.1 하단 네비게이션

| 탭 | 설명 |
|---|---|
| ToDo | 일반 할 일 관리 화면 |
| 시간 | 플레이스홀더 뷰(컨텐츠 숨김) |
| ReBorn | 캘린더 + 일정 리스트 + 휴지통 |
| 미룬 항목 | snooze 상태 항목 전용 목록 |
| 설정 | 플레이스홀더 뷰(컨텐츠 숨김) |

### 2.2 상단 영역

- 고정 헤더(스크롤 시 유지)
- ReBorn 로고 + 테마 토글
- 랜덤 동기부여 문구(중앙 정렬)

---

## 3. 기능 명세

### 3.1 할 일 기본 관리

| ID | 기능 | 상태 |
|---|---|---|
| F-01 | 할 일 추가(입력 + Enter/추가 버튼) | ✅ |

| F-03 | 할 일 삭제(확인 모달) | ✅ |
| F-04 | 완료/미완료 전환(체크박스 + 카드 클릭) | ✅ |

| A-03 | 서브태스크 추가/완료/삭제 | ✅ |

### 3.4 정렬/필터

## 4. 데이터 모델

```ts
type Todo = {
  id: string;
  title: string;
  completed: boolean;
  createdAt: string;         // ISO
  snoozeUntil?: string|null; // YYYY-MM-DD
  dueDate?: string|null;     // 시작일
  dueEndDate?: string|null;  // 종료일
  repeatDays?: number[];     // 0(일)~6(토)
  scheduleColor?: string|null;
  completedDates?: string[]; // 반복/기간의 일자별 완료 기록
  completedAt?: string|null; // 단일 일정 완료일
  subtasks?: { id: string; title: string; completed: boolean }[];
  tags?: string[];
  deletedAt?: string;        // 휴지통 이동 시각
};
```
