# ReBorn(ToDoApp) 웹 앱 명세서

## 1. 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | ReBorn (할 일 관리 웹 앱) |
| **목적** | 사용자가 할 일을 등록·관리·완료할 수 있는 웹 기반 할 일 관리 애플리케이션 |
| **대상** | 개인 사용자 (데스크톱·모바일 브라우저) |

---

## 2. 구현된 기능

### 2.1 할 일 관리 (핵심)

| 번호 | 기능 | 설명 | 상태 |
|------|------|------|------|
| F-01 | 할 일 추가 | 새 할 일 제목 입력 후 등록 (Enter 또는 추가 버튼) | ✅ |
| F-02 | 할 일 목록 조회 | 등록된 할 일 목록 표시, 정렬 옵션 지원 | ✅ |
| F-03 | 할 일 완료 처리 | 체크박스로 완료/미완료 전환 | ✅ |
| F-04 | 할 일 수정 | 더블클릭 또는 수정 버튼으로 제목 수정 | ✅ |
| F-05 | 할 일 삭제 | 단일 삭제 (삭제 확인 모달) | ✅ |

### 2.2 필터·정렬

| 번호 | 기능 | 설명 | 상태 |
|------|------|------|------|
| F-06 | 상태별 필터 | 전체 / 진행 중 / 완료 / 미룬 항목 | ✅ |
| F-07 | 정렬 | 최신순, 오래된순, 제목 가나다순, 제목 가나다역순 | ✅ |
| - | 태그 필터 | 태그별로 목록 필터 (전체 태그 화면 포함) | ✅ |

### 2.3 특별 기능

| 번호 | 기능 | 설명 | 상태 |
|------|------|------|------|
| - | 미루기 | 내일로 미루기 / 날짜 선택 후 해당 날 00시에 다시 표시 | ✅ |
| - | 꺼내기 | 미룬 항목 탭에서 즉시 목록으로 복귀 | ✅ |
| - | 서브태스크(항목 추가) | 할 일별 하위 항목, 전부 완료 시 할 일 자동 완료 | ✅ |
| - | 태그 | 할 일에 태그 추가·삭제, 제목 옆 #태그 표시 | ✅ |
| F-09 | 데이터 유지 | 로컬 스토리지로 새로고침 후에도 유지 | ✅ |
| F-10 | 완료 일괄 삭제 | 완료된 항목 일괄 삭제 (삭제 확인 모달) | ✅ |

### 2.4 UI·UX

| 항목 | 내용 | 상태 |
|------|------|------|
| 다크/라이트 모드 | 테마 토글, 시스템 설정 반영, 로컬 저장 | ✅ |
| 삭제 확인 | 할 일 삭제·완료 항목 삭제 시 커스텀 확인 모달 | ✅ |
| 미루기 확인 | 미루기 실행 전 커스텀 확인 모달 | ✅ |
| 동기부여 명언 | 푸터에 랜덤 짧은 명언 표시 | ✅ |
| 애니메이션 | 할 일 추가 시 등장 애니메이션, 모달 열릴 때 애니메이션 | ✅ |
| 브랜딩 | ReBorn 로고(깨진 달걀) + 제목 | ✅ |

---

## 3. 화면 구성 (UI)

| 화면/영역 | 구성 요소 |
|-----------|-----------|
| **헤더** | ReBorn 로고(깨진 달걀), 제목, 테마 토글, 설명 문구 |
| **입력 영역** | 입력 필드 + 추가 버튼 (Enter 제출) |
| **목록 영역** | 필터 탭(전체/진행 중/완료/미룬 항목) + 정렬, 태그 필터, 할 일 리스트 |
| **할 일 카드** | 체크박스, 제목, 추가 시간, #태그, 미루기/꺼내기, 수정/삭제, 항목 추가·태그 패널 |
| **푸터** | 남은 할 일 개수 + 동기부여 명언(배경 블록), 완료 항목 삭제 버튼 |

---

## 4. 데이터 모델 (할 일 아이템)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `id` | string | O | 고유 식별자 |
| `title` | string | O | 할 일 제목 |
| `completed` | boolean | O | 완료 여부 |
| `createdAt` | string | O | 생성 시각 (ISO 8601) |
| `snoozeUntil` | string \| null | - | 미루기 해제일 (YYYY-MM-DD) |
| `subtasks` | array | - | 서브태스크 목록 `{ id, title, completed }` |
| `tags` | string[] | - | 태그 목록 |

## 6. 기술 스택

| 구분 | 사용 |
|------|------|
| **프론트** | HTML + CSS + JavaScript (바닐라) |
| **저장** | 로컬 스토리지 |
| **스타일** | CSS 변수(다크/라이트), 반응형 |

---

## 7. 용어 정의

| 용어 | 설명 |
|------|------|
| **할 일** | 사용자가 등록한 한 건의 작업(제목, 완료 여부, 서브태스크, 태그 등) |
| **완료** | 사용자가 완료로 표시한 상태 |
| **진행 중** | 아직 완료로 표시하지 않은 상태 |
| **미룬 항목** | 미루기로 지정한 날이 되기 전까지 목록에서 숨겨진 항목 |
